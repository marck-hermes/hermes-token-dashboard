⚠️ Known Issues & Fixes (click to expand)

🕐 Ollama Timeouts

Symptom: ERROR timed out at ~295s (default 300s)

Cause: CPU-only 7B vision on 4 vCPUs, dense charts exceed 300s

Fix: STR_VISION_TIMEOUT=600 in .env

🚫 OpenRouter Rate Limits

Symptom: HTTP 429 on 2nd+ request, fallback to Ollama

Cause: Free tier ~3 req/min, burst ~1 req/10s

Mitigation: STR_VISION_OPENROUTER_DELAY=10, accept ~70% fallback

🔑 OpenRouter Auth

Symptom: HTTP 401 Missing Authentication header

Cause: .env has placeholder OPENROUTER_API_KEY=***

Fix: User edits /home/ubuntu/.hermes/profiles/str/.env with real key

⏱️ Cron Script Field

Symptom: "Script not found: cd ... && bash ..."

Cause: Cron script field expects filename only, not full command

Fix: script: "str_daily_pipeline.sh", workdir: "/home/ubuntu/.hermes/profiles/str/tv-shot"
