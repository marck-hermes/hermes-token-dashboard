📝 Notas Importantes

  • Rate limits Free Tier: ~20 req/min, ~1000 req/dia por modelo. Use fallback local (Ollama) para produção.
  • Preços: Valores de referência do OpenRouter; podem variar com cache/roteamento. Verifique openrouter.ai/models para preços atuais.
  • Context window: Modelos com 1M+ tokens (Gemini, MiniMax) custam mais por token de input em contextos longos.
  • Reasoning models (o1, r1, nemotron-3-ultra): Muito lentos (10-60s). Use apenas para planejamento, não para chat em tempo real.
  • Hermes Agent: Configure roles para roteamento automático (master→worker→fallback).
